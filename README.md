## Run Locally

Clone the project

```bash
  git clone https://github.com/GarvJain03/mailing-service.git
```

Go to the project directory

```bash
  cd mailing-service
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
